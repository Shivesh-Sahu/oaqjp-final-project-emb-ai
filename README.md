# Final Project - Emotion Detection Application

This project is part of the IBM Skills Network "Developing AI Applications with Python and Flask" course.

## Project Name
Emotion Detection Application

## Description
This web application uses the Watson NLP Emotion Detection service to analyze text and detect emotions such as:
- Joy
- Anger
- Fear
- Disgust
- Sadness

The application is built using Python and Flask and includes:
- Emotion detection module
- Flask web interface
- Unit tests
- Error handling
- Static code analysis
